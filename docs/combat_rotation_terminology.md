# Combat Rotation Terminology / 战斗轴术语对照

## 体系总览

战斗风格由两个独立维度组成：

```
配队类型（队伍结构）              操作层级（轴复杂度，连续光谱）
├── 主副C队 / Hypercarry          轮切 ◄──── 轮椅轴 ────► 合轴
├── 双C队 / Dual Carry            (朴素轮换)  (简单合轴)  (进阶合轴)
├── 3C互拐队 / Triple Carry
└── 效应队 / Synergy Team
```

- **配队类型**：决定队伍结构和角色分工，四种互斥分类
- **操作层级**：决定轴的复杂度和 DPS 上限，是连续光谱而非离散分类
- **轮椅轴**：不是独立种类，是光谱上偏简单的位置描述（用了部分合轴技巧但整体简单）
- 任意配队类型都可以在轮切↔合轴光谱上选择适合自己的操作层级

## 配队类型（队伍结构）

| 中文术语   | English Term            | 说明                                                                             |
| ---------- | ----------------------- | -------------------------------------------------------------------------------- |
| 主副C队    | Hypercarry              | 传统演奏队：3号位全队增幅，2号位专项增幅，1号位吃延奏输出，延奏期间不宜切人      |
| 双C队      | Dual Carry              | 一位通用辅助 + 两名C，两位C自由支配时间资源，通过合轴压缩双人输出流程            |
| 3C互拐队   | Triple Carry            | 三人同时承担输出+buff职责，每人的延奏对下一人有特定增益                           |
| 效应队     | Synergy / Effect Team   | 类似主副队（单人主要输出），但2号位 buff 无切人限制，流程自由度更高               |

## 操作层级（轴复杂度）

| 中文术语   | English Term            | 说明                                                                             |
| ---------- | ----------------------- | -------------------------------------------------------------------------------- |
| 轮切       | Sequential Rotation     | 一个角色打完整套动作再切下一个，不做后摇优化，朴素 ABC 顺序轮换                  |
| 轮椅轴     | (形容词，非独立种类)    | 用了部分合轴技巧但整体简单的轴，轮切→合轴光谱上的中间态                          |
| 合轴       | Quick Swap              | 大量运用 Swap Cancel，切人时前后角色动作重叠，双人同屏输出，达到 1+1 < 2 效果    |

> **Quick Swap** = 合轴打法（战斗风格）
> **Swap Cancel** = 合轴的核心操作技巧（切人取消动画，使两角色同屏输出）

## 操作层级特征对比

| 特征       | 轮切 (Sequential)               | 合轴 (Quick Swap)              |
| ---------- | ------------------------------- | ------------------------------ |
| 切人时机   | 整套动作打完再切                | 动作判定帧/伤害生效后立刻秒切  |
| 输出重叠   | 无，严格串行                    | 有，前后角色同屏同时输出       |
| 后摇处理   | 不处理，等后摇结束              | 用切人取消后摇                 |
| 操作难度   | 低，按顺序执行即可              | 高，需要精确时机判断           |
| DPS 效率   | 较低，有空窗期                  | 高，时间压缩                   |

## 操作技巧术语

| 中文术语     | English Term              | 说明                                                                                |
| ------------ | ------------------------- | ----------------------------------------------------------------------------------- |
| Swap Cancel  | Swap Cancel               | 合轴的核心技巧：技能释放后动画未结束时切人，前一角色动画继续执行，两角色同屏输出    |
| 取消后摇     | Animation Cancel          | 用闪避/大招动画/切人打断技能的后摇帧，减少无效等待时间                              |
| 脱手技能     | Detach Skill / Off-field  | 技能判定/伤害生成后即可打断，打断不影响伤害效果，判定节点远早于动作完成节点          |
| 轴长         | Rotation Length           | 一轮完整流程所需时间，下限由队伍中最长 CD 技能（通常是大招）决定                    |
| 循环轴       | Rotation Loop             | 队伍可重复执行的最小流程单元                                                        |
| 启动轴       | Opener / Startup Rotation | 首轮特殊流程，利用开局满大招能量和技能 CD 就绪的优势弥补首轮协奏从零开始的 DPS 亏损 |
| 透支         | Forte/Energy Overdraft    | 在资源未攒满时提前消耗（如先放技能再补能量），提高资源利用率                         |
| 拖轴         | Rotation Delay            | 操作失误或被打断导致流程延长，DPS 下降                                              |
| 暖机         | Ramp-up / Pre-buff        | 角色在正式输出前先释放 buff 类技能为后续输出做准备                                  |

## 合轴示例：卡夏千 (Carlotta-Xiangli Yao-Chixia) 17秒启动

### 启动轴
1. 千咲起手，跳下落攻击切夏空
2. 夏空下落攻击+A出一格能量，看到能量后E+声骸秒切千咲
3. 千咲E+A出大剪刀，看到剪刀切夏空
4. 夏空跳下落A（两格能量），再跳下落A（三格），闪避重击金光大，变奏千咲
5. 千咲声骸大招，强化E秒切卡提打平A
6. 卡提A出方格后放E，秒切千咲
7. 千咲点A（不要长按A），A出跳电锯终结满协奏切卡提
8. 卡提变奏入场（3剑），直接跳下落打三剑，开大

### 循环轴
1. 大卡提正常打结束，切变奏夏空得1格，A2下得第二格，E+声骸切千咲
2. 千咲E+A出大剪刀切夏空
3. 夏空跳下落出第三格，闪避重击金光大，切千咲
4. 千咲声骸大招，强化E秒切卡提打平A
5. 卡提A出方格后放E，秒切千咲（卡提E可以慢放一点避免吞剑）
6. 千咲点A（不要长按A），A出跳电锯终结满协奏切卡提
7. 卡提变奏入场（3剑），直接跳下落打三剑，开大
8. → 回到步骤1循环

> 特征：全程"秒切"，利用切人取消后摇，前后角色输出窗口大量重叠。
